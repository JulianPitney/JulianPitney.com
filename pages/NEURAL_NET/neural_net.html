Neural Network

Project Description

This project was inspired by curiosity about neural networks. Sometime in 2016, I decided to teach myself how gradient descent, back-propagation and basic neural networks work. I did this by implementing my own from scratch (using just the C++ standard library) and then convincing myself it works by training it on the MNIST dataset and having it accurately predict the digits in the test dataset.

Gradient Descent

Neural nets work by defining an error function that compares the actual output of the network to the desired output of the network. The goal is to minimize this error function. Gradient descent works by findinding the partial derivative of the error function with respect to each variable that the error function depends on. These derivatives give the direction (sign) of the rate of change of the error function with respect to each variable, showing you whether you need to increase or decrease that variable in order to reduce the value of the error function. This increase or decrease is done in small increments, once each iteration, to avoid overshooting the adjustments. The complex part is that nested within the error function is the output of the network, and the output of the network is itself a composite function consisting of the activation function of every node in the network (which are all linked together and feeding forward into each other). So the error function is a composite of thousands to billions of functions, which makes finding it's partial derivatives tricky. In addition to that, you have to find the partial derivative of the error function once for every variable in that function (which includes every weight and bias in every layer of the network). Only then can you update all the weights and biases in the correct direction, reducing the overall error for that input. This whole procedure is repeated once for every training input. This allows us to "descend the gradient" of the error function. This is the fundamental algorithm that makes neural networks work, which as we know in 2025, has turned out to be a really powerful idea.
