Project Overview

This project involved building a scalable, cloud-based platform to modernize and optimize clinical trial operations for a healthcare company. The goal was to replace outdated systems with automated workflows, high-quality data management, and advanced analytics, ensuring compliance with regulations like PHIPA, PIPEDA, and Law 25.

Key Objectives

  • Automate workflows and digitize data
  • Enhance data quality and implement robust error detection
  • Build secure, compliant cloud infrastructure
  • Streamline audits and access controls
  • Enable real-time analytics and scalability for future growth
  • Leverage AI tools to improve efficiency and patient experience

My Contributions

  • Led a team of 5 developers to deliver the platform
  • Designed and deployed cloud infrastructure using AWS, Kubernetes, and Terraform
  • Built CI/CD pipelines to test, build and deploy to multiple environments
  • Developed a robust data model and implemented it in Django+Postgres
  • Explored the LLM revolution and built A.I agents using Langgraph
  • Oversaw cybersecurity audits and ensured regulatory compliance

Technology Stack

Python Django Rest Framework AWS Kubernetes Terraform PostgreSQL React Docker OpenAI API Langgraph
